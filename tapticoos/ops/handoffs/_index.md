# Handoffs Index

Generated `#war-room` packets live in this directory, one file per handoff, named `YYYY-MM-DD-<client_or_internal>-<slug>.md`.

## Conventions

- **Naming**: `2026-05-07-acme-q3-pixel-setup.md` — date, client (or `internal`), short slug
- **Template**: `_template.md` — every packet follows it section by section
- **Inbox**: `inbox/` (gitignored) — where Cowork exports land before being converted to packets
- **Generator**: `bo-handoff-preparer` subagent or `/handoff` slash command

## Lifecycle

1. Source material lands in `inbox/` (or is referenced directly from a task ledger entry)
2. `cowork-to-war-room.sop.md` runs → packet generated here, ledger updated to `state: handoff`
3. Bo executes, edits Section 9 (completion report) in place, saves
4. `memory-compile.sop.md` runs → packet content compiled into `memory/wiki/`, ledger flips to `completed`
5. Packet stays in this directory as the audit record. Do not delete.
